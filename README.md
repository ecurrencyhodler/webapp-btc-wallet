# LedgerBTC
A bitcoin wallet webapp built using Ledger's device signer kit (https://www.npmjs.com/package/@ledgerhq/device-signer-kit-bitcoin). 

## Features
- USD balance
- Send
- Receive with address verification
- Message signing + message verification
- Links to mempool.space for transactions
